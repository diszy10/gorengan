# Mendoan
API service using Go

## Requirement

- Go 1.16+
- PostgreSQL 13+

## Prerequisite
```sh
# Install dependencies
$ go get
```

Copy *.env.example* to *.env* and fill the variable accordingly

## Run Server
```sh
$ go run .
```
